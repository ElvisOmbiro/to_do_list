Features

To-Do List

A to-do list application that is used to write down notes that can store and delete memory on command, and that can be private.

About

This to-do list app is designed to help you manage your tasks with a focus on privacy and simplicity. You can easily add, delete, and retain tasks, making it a versatile tool for personal and professional use.

The project was inspired by my habit of taking a lot of notes and my desire for an easy way to take notes, which this application enables.
